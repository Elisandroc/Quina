Gerador de dezenas da Quina para Arduino<br/>
Itens:<br/>
Display oled SH1106 SPI Resolução 128×64;<br/>
Arduino Uno;<br/>
Botão temporário simples;<br/>

Conexões:<br/>
<pre>
Pino do Display	Pino do Arduino<br/>
VCC	            | 5V (ou 3.3V, dependendo do display!)
GND	            | GND
SCL / CLK	    | D13
SDA / MOSI	    | D11
RES	            | D9
DC	            | D8
CS	            | D10 
<br/>
Botão
Pino 1          | GND
Pino 2          | Pino 2</pre>
